Dynamic Pathfinding Agent

Requirements:

Python 3.x

Run:

python main.py


Features:

- A* Search
- Greedy Best First Search
- Manhattan Heuristic
- Euclidean Heuristic
- Dynamic Obstacles
- Replanning
- Interactive Grid


Controls:

Click = Add/remove wall

Generate Maze = Random map

Start Search = Run algorithm

Dynamic Mode = Enable moving obstacles
